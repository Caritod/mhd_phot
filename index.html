HUACHO-MHD CODE.

The Huacho-MHD code project is a grid-based code for astrophysical magnetohydrodynamics (MHD). This code includes a variety of numerical algorithms, such as different Riemann solvers and spatial reconstruction:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

The current version implements algorithms for the following physics:

photoionization passive scalars advected with the flow, static gravitational potential, both isotropic and anisotropic thermal conduction, optically-thin radiative cooling.

In addition following grid and parallelization options are avilable:

--cartesian coordinates     
--parallelization using domain decomposition and  MPI.
